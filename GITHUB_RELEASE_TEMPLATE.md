# Junktoys v1.0 - Ultimate Windows Optimizer 🧹

A powerful Windows debloating and optimization tool with advanced registry tweaks, auto dark mode, and one-click optimizations.

## 📥 Download

**[Download Junktoys.exe](#)** (~75 MB) - Single portable EXE with icon!

## ✨ Features

### Basic Tools
- ⚡ **Quick Clean** - One-click optimization (kills heavy apps + clears temp files)
- **Background Apps Manager** - Auto-kill processes using >500MB RAM
- **Bloatware Detector** - Remove pre-installed apps & Edge
- **Startup Manager** - Control auto-start programs
- **Services Optimizer** - Disable unnecessary services

### 🚀 Advanced Optimizations
- **Full Optimization Pack** - Apply all tweaks with one click
- **Disable Telemetry** - Stop Windows data collection completely
- **Disable Cortana** - Remove permanently
- **Ultimate Performance Power Plan** - Unlock hidden Windows mode
- **Network Optimization** - Low-latency gaming tweaks
- **SSD Optimization** - TRIM command
- **Privacy Controls** - Disable tracking, location, advertising ID
- **Visual Effects** - Optimize for maximum performance
- **Telemetry IP Blocking** - Block Microsoft tracking servers

### UI Features
- **Auto Dark Mode** - Automatically matches your Windows theme
- **Modern Interface** - Clean, easy-to-use design
- **Fast Performance** - Optimized build

## 🚀 Quick Start

1. **Download** `Junktoys.exe`
2. **Right-click** → **Run as administrator**
3. Use **⚡ Quick Clean** for instant optimization
4. Explore **⚡ Advanced** tab for powerful tweaks

**No installation, no extraction, no dependencies!** Just download and run.

## 📋 Requirements

- Windows 10 version 1809+ or Windows 11
